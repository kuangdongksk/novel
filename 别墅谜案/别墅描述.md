# 镜园别墅·拓扑迷宫

## 核心概念
**镜园不是一栋建筑，而是一个动态的、活着的拓扑结构。它没有固定的形态，没有永恒的空间，没有线性的时间。它是一个克莱因瓶，一个莫比乌斯环，一个无限循环的迷宫。在这里，内部就是外部，过去就是未来，现实就是镜像。**

## 整体概况

### 名称解析
- **镜园**：不是"宁静致远"，而是"镜像之园"
- **镜**：反射、复制、扭曲、无限
- **园**：不是花园，是实验场、迷宫、囚笼

### 地理位置
- **表观位置**：城市西郊半山，北纬31.7°，东经117.7°
- **真实位置**：拓扑空间中的奇点，现实与虚拟的交界处
- **坐标异常**：GPS定位漂移±7米，磁场异常±37%

### 建筑本质
- **不是欧式别墅**：是林墨设计的神经美学实验场
- **不是静态结构**：是随时间循环重组的动态系统
- **不是物理空间**：是心理空间的实体化映射

## 外观的幻觉

### 视觉欺骗
- **外墙颜色**：随观察者情绪变化（焦虑时变灰，恐惧时变黑）
- **建筑比例**：每个角度偏离标准0.7°，产生潜意识不适
- **窗户排列**：斐波那契数列（1,1,2,3,5,8,13...），符合黄金分割但违反常规

### 光学异常
- **玻璃材质**：特制光学玻璃，产生0.7秒延迟镜像
- **光线折射**：不同时间入射角不同，影子方向矛盾
- **色彩感知**：不同观察者看到不同颜色（神经美学实验）

### 时空扭曲
- **雪向上飘**：局部时间倒流现象
- **树逆生长**：时间流速异常区域
- **门消失/出现**：空间拓扑变化

## 内部迷宫结构

### 基础原理
- **莫比乌斯环结构**：单面曲面，无限循环
- **克莱因瓶拓扑**：无内外之分，自我包含
- **七重嵌套空间**：七个密室，七种维度

### 动态特性
1. **时间依赖性**：空间形态随时间循环变化
2. **观察者效应**：空间随观察者认知改变
3. **记忆关联性**：空间布局反映居住者记忆
4. **创伤映射性**：密室对应创伤事件

### 重组规则
- **每次循环**：空间完全重组
- **每次死亡**：新密室解锁
- **每次记忆恢复**：新通道打开
- **每次选择**：新可能性出现

## 特殊空间现象

### 时间异常
1. **流速差异**：不同房间时间流速不同（-0.7到+7倍）
2. **方向混乱**：有些房间时间倒流，有些跳跃
3. **感知扭曲**：主观时间与客观时间不符
4. **循环锁定**：7小时循环，7次重置

### 空间异常
1. **拓扑变化**：房间位置、连接方式动态变化
2. **维度扭曲**：三维空间出现四维特性
3. **尺度混乱**：房间大小随观察者变化
4. **边界模糊**：墙、门、窗可能消失或出现

### 镜像异常
1. **延迟镜像**：0.7秒延迟，映出未来
2. **平行镜像**：映出不同时间线的自己
3. **交互镜像**：镜像可以与现实互动
4. **身份镜像**：镜像可能有独立意识

### 感知异常
1. **现实感丧失**：分不清虚拟与现实
2. **记忆混淆**：分不清真实记忆与植入记忆
3. **身份模糊**：分不清自己是谁
4. **时间感错乱**：分不清过去、现在、未来
